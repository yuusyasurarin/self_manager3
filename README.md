# self_manager3
学習しながら作るアプリ
JavaScriptを使用してUIを作成することをこのアプリを通じて学ぼうと考えている。PythonやC言語も学習する必要があるので違いなどを意識しながら少しずつ学習していく。
## アプリ作成記録
### **3月31日**  
- このリポジトリを作成した。今日から毎日少しずつアプリの作成を通じて学習していこうと思う。
- JavascriptでWebサイトに関わるたいていのことは実現することができる。
- GitHub上で最も多く使用されたプログラミング言語の第1位がJavaScript（pythonは2位、C++は5位）
- HTML(素材）,CSS(味付け),JavaScript(調理)でセットになっている。
- フロントエンド：ユーザーの目につく部分、操作の開発。HTML,CSS,JavaScriptが必須になる。
- バックエンド：API提供やセキュリティ確保、データ処理など。(JavaScript,Python,Ruby,PHP,Goなど）
- **JavaScriptはChromeやFirefoxなどのブラウザ上で動作する**プログラミング言語。普通のPythonやC言語などはパソコン上で動作するもの。
- 本来ブラウザ上で稼働するJavaScriptをパソコン上（OS上）で動作できるようにするためのものがNode.jsという実行環境。Pythonを初めて使用するときにWebからPythonの言語のパーッケージをダウンロードしたときのようなもの。
#### HTML,CSS,JavaScriptの役割
-　**HTML**はWEBサイトのパーツや骨格を作成する。（見出し、文字、ボタンなど）。人間の目でボタンを認識できるようにする。
  ```
  <botton>もっと見る</button>
  ```
様々なタグが存在している。それぞれ調べて学習する。
